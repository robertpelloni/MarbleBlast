# AGENT INSTRUCTIONS

- This project is a TypeScript web game using custom rendering and physics.
- The build process uses `rollup`. To compile: `npm run compile`. To watch: `npm run watch` or `npm run watch-fast`.
- For any new features, please comment code in depth regarding what it's doing, why it's there, side effects, bugs, optimizations, alternate methods, etc.
- Only comment if it's not self-explanatory.
- Always document input information in comprehensive, thorough detail.
- If you update a version number, ensure the CHANGELOG.md is updated, and perform a git commit/push with the version bump referenced in the commit message.
- Always verify all changes locally and ensure they don't break existing functionality (especially physics).
- Reference the current state of features in TODO.md and ROADMAP.md.
